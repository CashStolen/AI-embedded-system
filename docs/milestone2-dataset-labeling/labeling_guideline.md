# Labeling Guideline (Ordinal 0..4)

## Label Definition
0 very_blurry: iris/eyelash boundary not distinguishable
1 blurry: boundary barely visible
2 ok: boundary visible but not crisp
3 sharp: crisp boundary, good texture detail
4 very_sharp: very clear texture, high edge contrast

## Notes
- Over/under-exposure that removes detail should be labeled lower.
- Use consistent viewing scale (100%) when reviewing.
